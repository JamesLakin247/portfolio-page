Blog App

Functionality

Users can create an account, login, and logout

Once logged in, users can create and delete their own blog posts

Users can see other peoples posts regardless of whether they are logged in or not

About

Built with HTML, CSS, JavaScript React.js, Redux Toolkit, Node.js, Express, and MongoDB

Users accounts and their posts are stored on MongoDB and this data can be accessed by the frontend using the REST API I have created in Node.js

User passwords are stored as a hashed value on the database
